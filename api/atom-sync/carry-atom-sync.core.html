carry-atom-sync.core

add

(add blueprint r)

On :on-start signal, after wrapped app is started, creates a bidirectional sync between the specified atom-like reference and the app model and also updates the reference from the model for the first time. The initial reference value is not used. Sync is removed on :on-stop signal.
